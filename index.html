LINGUAGEM DE PROGRAMACAO

NetBeans

O NetBeans é a plataforma em que será desenvolvido as aplicações em Java. É ele que será usado para escrever as linhas de codigos e rodar os aplicativos.

Baixar

JDK

O JDK é um componente para o NetBeans, é um tipo de biblioteca de aplicações que será usado pelo NetBeans para permitir a programação.

Baixar

Como instalar?

Caso não saiba instalar o NetBeans ou o JDK, não tem problema. Aqui abaixo terá um tutorial em video bem rápido e objetivo para te ajudar.

Jvdroid - IDE for Java

Para aqueles que não possui computador, este app é um quebra-galho. Com ele é possivel programar em java, dá pra aplicar bastante coisa que o professor ensinar, apesar de ter algumas limitações.

PROGRAMACAO PARA INTERNET

Notepad++

O Notepad++ é um editor de texto universal, que pode abrir a maioria dos formatos de arquivos de texto. No curso, será usado para escrever arquivos HTML gerando uma pagina de internet. Extremamente recomendado, pois será usado nas aulas pelo professor.

Baixar

Visual Studio Code
(opcional)

O Visual Studio Code é também uma plataforma de programação, assim como o Netbeans. Eu o recomendo apenas para quem tem realmente interesse em programar e quer ir mais além na área. Foi com ele que criei este site.
